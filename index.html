Experiencia

Desarrollé un sistema integral para gestionar ventas, inventarios, clientes y usuarios. Este sistema incluye funciones como cuadre de caja, opciones de análisis de ventas, crédito para clientes conocidos tipo fiado, y una opción para marcar la mejor ruta y el orden de entrega para los domicilios. Utilicé Python junto con librerías como Pandas, NumPy, SciPy, GeoPandas y Matplotlib. La interfaz fue creada con Streamlit, y la base de datos se gestionó con PostgreSQL, desplegada en Render.

Creé un chatbot capaz de interpretar el lenguaje natural y responder preguntas específicas relacionadas con un negocio. Este chatbot se diseñó para integrarse en diversas plataformas empresariales, ofreciendo respuestas precisas y útiles. El proyecto se desarrolló usando Python, Streamlit y la API de OpenAI para la interpretación del lenguaje natural.
